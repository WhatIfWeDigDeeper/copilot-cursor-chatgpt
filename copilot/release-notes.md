## [October 2025](https://code.visualstudio.com/updates/v1_106)


### [Save conversation as prompt](https://code.visualstudio.com/updates/v1_106#_save-conversation-as-prompt)

> You can now save your chat conversations as reusable prompts with the /savePrompt command.


### [Plan agent](https://code.visualstudio.com/updates/v1_106#_plan-agent)

> A new plan agent helps developers break down complex tasks step-by-step before any code is written. Select Plan from the agents dropdown in the Chat view to get started.

Remember, you could use the `/savePrompt` command here.
